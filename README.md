# vue_shop

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```

### Customize configuration

See [Configuration Reference](https://cli.vuejs.org/config/).

### 文件结构说明

```
├─public          // 公共部分
└─src
    ├─assets
    │  └─css      // 全局CSS
    ├─components  // 页面组件
    │  ├─goods    // 商品组件
    │  ├─order    // 订单组件
    │  ├─power    // 权限管理
    │  ├─report   // 报表组件
    │  └─user     // 用户组件
    ├─plugins     // 插件
    └─router      // 路由
```
